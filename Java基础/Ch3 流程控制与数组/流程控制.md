# 流程控制



## 分支语句

### if语句

```java
if(logic expression){}
//-------------------------------
if(logic expression){
    
}else{}
//-------------------------------
if(logic expression){
    
}
else if(logic expression){
    
}else{}
```

### switch语句

```java
switch(expression/value){// 可以是byte short char int Stirng 枚举类型
    case value1:{
        // to do
        break;
    }
    case value2:{
        // to do
        break;
    }
    default{
        // to do
    }
}
```



## 循环结构

### while循环语句

```java
while(boolean expression){
    // to do
}
```

### do while循环语句

```java
do{
    // to do
}while();
```

### for循环

```java
for([init_statement];[test_expression];[iteration_statement]){
    
}
```
